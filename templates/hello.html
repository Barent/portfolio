About

I am a software developer based out of Austin Texas specializing in full stack JavaScript. Currently I work on a large code base of C# and JavaScript for a deployment of Dynamics CRM. Most of my projects have been built using meteor but I also enjoy working with node and flask. This site is served up using flask and I hope to add a blog seciton that utilizes react in the future.

When I am not programming I like to spend time with my fiance and dog. We go on hikes when we can and enjoy spending time in the water when the weather permits. I have moved around a lot. I grew up in Pennsylvania and when I was in my early twenties I moved to Colorado. I spent a lot of time out west and lived in both Colorado and Wyoming. I now live in Austin Texas and have been loving eternal summer.

Open Source Projects

These projects were built using full stack JavaScript.

Research-Helper

This is an applicaiton to help with research collection. The idea was spawned after I helped my fiance with collecting inventory at her job. The applicaiton I built for her was called research-inventory and was hosted on Meteor's free hosting service. This version is more general and intended to be used by any field researcher.

Cedar-CRM

This is a small CRM applicaiton build using Meteor kitchen. It has three main sections and contains excell export. The application still needs joins and I have created a demo version with joins and it is currently in the github repo for this project.

Simple-Scheduler

This is an application to help my future father in laws small business schedule field techs. It has a user account system with varrying roles and is invite only. The calendar piece was built with the jQuery full calendar. This is stil in progress
